# WEBSITE LAUNCH CHECKLIST
# LOCAL SERVICE BUSINESS WEBSITES

Purpose:
Ensure every website is:
- production ready
- SEO optimized
- mobile optimized
- conversion optimized
- technically sound
- ready for indexing

Complete ALL checks before launch.

---

# 1. GENERAL SITE REVIEW

Verify:
- all pages are complete
- branding is consistent
- layouts are polished
- no placeholder content exists
- no lorem ipsum remains
- no broken sections exist

Ensure:
- website feels trustworthy
- website feels professional
- messaging is clear

---

# 2. MOBILE RESPONSIVENESS

Test:
- iPhone sizes
- Android sizes
- tablets
- desktop widths

Verify:
- no horizontal scrolling
- buttons are tappable
- spacing is consistent
- text is readable
- navigation works properly

Check:
- sticky headers
- mobile menus
- CTA visibility
- forms on mobile

---

# 3. PAGE SPEED & PERFORMANCE

Run:
- Lighthouse audit
- PageSpeed Insights

Target:
- Lighthouse score 90+

Verify:
- images optimized
- lazy loading enabled
- no oversized assets
- minimal JS bundles
- no layout shifts

Avoid:
- autoplay videos
- giant animations
- unused scripts

---

# 4. SEO VALIDATION

Verify every page has:
- unique title tag
- unique meta description
- proper H1
- logical heading structure
- canonical URL
- Open Graph tags
- Twitter tags

Ensure:
- URLs are clean
- keywords are natural
- no duplicate metadata exists

---

# 5. LOCAL SEO CHECKS

Verify:
- city/service area mentions exist
- NAP data is consistent
- service pages exist
- location pages exist
- internal links work

NAP =
- Name
- Address
- Phone Number

Ensure consistency across:
- website
- Google Business Profile
- directories

---

# 6. GEO / AI SEARCH OPTIMIZATION

Verify content clearly explains:
- what the business does
- where they operate
- who they help
- how customers contact them

Ensure:
- headings are descriptive
- FAQs exist
- content is easy to skim
- wording is concise

Optimize for:
- ChatGPT
- Google AI Overviews
- Perplexity
- voice search

---

# 7. SCHEMA VALIDATION

Verify:
- JSON-LD is valid
- schema matches visible content
- LocalBusiness schema exists
- Service schema exists
- FAQ schema exists when applicable

Test:
- Google Rich Results Test
- schema validators

Ensure:
- no schema conflicts
- no fake review schema

---

# 8. ACCESSIBILITY CHECKS

Verify:
- sufficient contrast
- alt text exists
- forms are labeled
- keyboard navigation works
- semantic HTML used properly

Ensure:
- buttons are accessible
- focus states are visible
- headings are structured logically

---

# 9. FORM TESTING

Test:
- contact forms
- quote forms
- booking forms

Verify:
- submissions work
- emails send properly
- success messages display
- spam protection works

Ensure:
- mobile usability
- required fields work correctly

---

# 10. CTA & CONVERSION REVIEW

Verify:
- CTAs visible above fold
- phone number easy to find
- contact methods obvious
- trust indicators present

Check:
- testimonials
- reviews
- guarantees
- response expectations

Ensure:
- friction is minimized
- forms are short and simple

---

# 11. IMAGE REVIEW

Verify:
- images compressed
- alt text added
- responsive sizing works
- no blurry images
- no stretched images

Prefer:
- webp format
- real local photos

Avoid:
- giant image files
- cheesy stock photos

---

# 12. INTERNAL LINKING

Verify:
- service pages linked
- city pages linked
- navigation works
- footer links work

Ensure:
- no orphan pages
- descriptive anchor text used

Avoid:
- generic "click here" links

---

# 13. CONTENT REVIEW

Verify:
- copy sounds human
- no AI fluff
- no spelling mistakes
- no grammar issues
- paragraphs readable
- headings descriptive

Ensure:
- trust is reinforced
- benefits are clear
- CTAs are strong

---

# 14. TECHNICAL QA

Verify:
- no console errors
- no broken links
- favicon works
- sitemap.xml exists
- robots.txt exists

Ensure:
- HTTPS enabled
- redirects working properly
- deployment successful

---

# 15. ANALYTICS & TRACKING

Verify installation of:
- Google Analytics
- Google Search Console
- Meta Pixel if applicable

Ensure:
- tracking scripts working
- conversions track properly

Optional:
- call tracking
- heatmaps
- event tracking

---

# 16. SOCIAL SHARING CHECKS

Verify:
- Open Graph images work
- social previews look correct
- titles/descriptions display properly

Test:
- Facebook sharing debugger
- LinkedIn previews

---

# 17. SECURITY & RELIABILITY

Verify:
- forms protected from spam
- dependencies updated
- environment variables secured
- backups available

Ensure:
- no exposed secrets
- production configuration correct

---

# 18. FINAL USER EXPERIENCE REVIEW

Ask:
- Is the website easy to understand?
- Is it obvious what the business does?
- Is contacting the business effortless?
- Does the site feel trustworthy?
- Does the mobile experience feel polished?

The site should:
- feel professional
- feel modern
- feel fast
- feel trustworthy

---

# 19. PRE-INDEXING CHECKS

Before indexing verify:
- noindex tags removed
- sitemap submitted
- robots.txt correct
- canonical tags correct

Ensure:
- pages are crawlable
- important pages linked internally

---

# 20. POST-LAUNCH TASKS

After launch:
- submit sitemap to Google Search Console
- monitor indexing
- monitor Lighthouse scores
- monitor form submissions
- test site on real devices
- monitor rankings

Recommended:
- request Google reviews
- create GBP posts
- build citations/directories

---

# FINAL LAUNCH APPROVAL

The website is ready when:
- mobile experience is excellent
- SEO is optimized
- local SEO is implemented
- GEO optimization is implemented
- forms work properly
- performance is fast
- trust signals are present
- conversion flow is strong
- no technical issues remain

The final website should:
- rank locally
- convert visitors
- load quickly
- feel premium
- work perfectly on mobile
- be easy for AI systems to understand
