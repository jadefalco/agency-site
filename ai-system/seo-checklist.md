# SEO CHECKLIST
# LOCAL SERVICE BUSINESS WEBSITES

Before completing any website or page, verify ALL items below.

---

# 1. PAGE TITLES

Each page must have:
- unique title tag
- primary keyword near beginning
- city/location when relevant
- natural wording
- under ~60 characters when possible

Good examples:
- Kelowna Plumbing Services | ABC Plumbing
- Emergency Electrician in Vernon | XYZ Electric

Avoid:
- generic titles
- duplicate titles
- keyword stuffing

---

# 2. META DESCRIPTIONS

Each page must:
- have unique meta description
- summarize page clearly
- include CTA when appropriate
- include keyword naturally
- stay under ~155 characters when possible

Good examples:
- Trusted Kelowna plumbers offering emergency repairs, installations, and fast service. Call today for a free estimate.

Avoid:
- duplicate descriptions
- keyword stuffing
- vague marketing language

---

# 3. URL STRUCTURE

URLs should be:
- short
- readable
- lowercase
- hyphen-separated

Good:
- /services/plumbing
- /kelowna-electrician

Bad:
- /page?id=12
- /SERVICES123

---

# 4. HEADING STRUCTURE

Each page should:
- have one primary H1
- use logical H2/H3 structure
- include keywords naturally

Avoid:
- multiple competing H1s
- skipped heading levels
- headings only for styling

---

# 5. LOCAL SEO

Ensure pages include:
- city/service area mentions
- local relevance
- nearby service areas
- local trust indicators

Include:
- service areas section
- Google Map if appropriate
- NAP consistency

NAP =:
- Name
- Address
- Phone Number

---

# 6. SCHEMA MARKUP

Include structured data when relevant:
- LocalBusiness
- Service
- FAQPage
- Review
- BreadcrumbList

Verify:
- valid JSON-LD
- accurate business info
- matching NAP data

---

# 7. INTERNAL LINKING

Each page should:
- link to related services
- link to contact page
- link to relevant city pages
- use descriptive anchor text

Avoid:
- orphan pages
- generic "click here"

---

# 8. IMAGE SEO

All images should:
- use descriptive filenames
- include alt text
- be compressed
- use responsive sizing

Prefer:
- webp format
- next/image
- lazy loading

Avoid:
- giant images
- missing alt text
- IMG_4837.jpg filenames

---

# 9. PAGE SPEED

Verify:
- fast mobile load speed
- optimized assets
- minimal JS
- no layout shifts

Target:
- Lighthouse 90+

Avoid:
- autoplay video
- giant background videos
- excessive animation libraries

---

# 10. MOBILE OPTIMIZATION

Verify:
- readable text sizes
- large tap targets
- visible CTA buttons
- no horizontal scrolling

Important:
- phone number clickable
- forms usable on mobile

---

# 11. CONTENT QUALITY

Content should:
- answer user intent
- be easy to skim
- use plain English
- sound human

Use:
- FAQs
- bullets
- short paragraphs
- clear CTAs

Avoid:
- fluff
- keyword stuffing
- AI-sounding filler

---

# 12. GEO (AI SEARCH OPTIMIZATION)

Content should clearly answer:
- what the business does
- where they operate
- who they help
- why they are trusted
- how customers contact them

Use:
- concise answers
- entity-rich wording
- structured sections
- FAQ content

Optimize for:
- ChatGPT
- Google AI Overviews
- Perplexity
- voice search

---

# 13. TRUST SIGNALS

Include when possible:
- reviews
- testimonials
- certifications
- years in business
- guarantees
- local references

Avoid:
- fake reviews
- exaggerated claims

---

# 14. ACCESSIBILITY

Verify:
- sufficient contrast
- form labels
- keyboard navigation
- semantic HTML
- alt text

---

# 15. CONVERSION OPTIMIZATION

Every page should:
- have a clear CTA
- reduce friction
- make contact easy

Include:
- visible phone number
- contact form
- quote button
- trust indicators

---

# 16. TECHNICAL SEO

Verify:
- canonical tags
- sitemap.xml
- robots.txt
- Open Graph tags
- Twitter tags

Ensure:
- no broken links
- no console errors
- no duplicate metadata

---

# 17. FINAL PRE-LAUNCH CHECK

Before launch:
- test mobile devices
- test forms
- test navigation
- verify metadata
- run Lighthouse audit
- verify schema
- check page indexing readiness
- test page speed

---

# SUCCESS CRITERIA

The website should:
- load fast
- rank locally
- convert visitors
- look trustworthy
- work perfectly on mobile
- be easy for AI systems to understand
